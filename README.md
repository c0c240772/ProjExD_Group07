# こうかまん
![こうかまん](fig/screen_shot.png)

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* パックマンみたいなゲーム
* こうかとんを操作して敵から逃げながらクッキーを全て食べるゲーム
* 参考URL：[プログラミング教室テクノロ パックマン](https://programming-school-technolo.com/?page_id=1783)

## ゲームの遊び方
* 移動：十字キー
* 周囲のクッキー吸い込み：Sキー(1回限り)
* ダッシュ：Dキー(300点消費)
* 壁を飛び越える：Fキー(3回まで)
* 中央左右のトンネルに入ると逆側の出口に出る
終了条件：クッキーを全て食べる、敵に当たる
スコア：クッキーを１つ食べるごとに１０点増加

## ゲームの実装
### 共通基本機能
* 敵味方の見た目変更
* 敵の挙動変更
* マップ、キャラクターの描画

### 分担追加機能
* 時を止める（担当：工藤）：アイテムを取ったら敵が停止
* 左右のワープ（担当：松岡）：端に行ったらワープ
* こうかとんの壁飛びスキル（担当：大友）：Fを押したとき発動
* 吸い込みスキル追加（担当：渡辺）：Sキーを押したら発動
* ダッシュ（担当：佐伯）：Dを押しながら操作

### 時を止める
* アイテムの時計を取ると敵の動きが止まる
* マップの固定場所にアイテムが湧く

### 飛行スキル
**Fキーを押すことで前方2マス先に飛行**
* 飛行先が壁だった場合壁がなくなる場所まで飛行
* 三回まで使用可能：残り回数は左下に表示

### 吸い込み
* 発動方法:Sキーを押すと発動
* 効果:7x7マスの範囲にあるクッキーを一気に吸い込んで獲得
* 1回のみ使用可能な必殺技で、使用時は水色のエフェクトが表示。

### ダッシュ
* 発動方法：Dキーを押しながら移動
* 300点消費

### メモ
* 実装したかった機能：敵の挙動を時々自機狙いにする
