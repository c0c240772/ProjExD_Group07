# こうかまん

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* パックマンみたいなゲーム
* 参考URL：[プログラミング教室テクノロ パックマン](https://programming-school-technolo.com/?page_id=1783)

## ゲームの遊び方
* 矢印キーでこうかとんを操作敵に触れたらゲームオーバー

## ゲームの実装
### 共通基本機能
* 敵味方の見た目変更
* 敵の挙動変更

### 分担追加機能
* タイトル追加
* パワーアップアイテムの追加（担当：）
* 左右のワープ（担当：）
* こうかとんの壁飛びスキル（担当：）
* 吸い込みスキル追加（担当：）

### 飛行スキル
**Fキーを押すことで前方2マス先に飛行**
* 飛行先が壁だった場合壁がなくなる場所まで飛行
### メモ
* クラス内の変数は，すべて，「get_変数名」という名前のメソッドを介してアクセスするように設計してある
* すべてのクラスに関係する関数は，クラスの外で定義してある
